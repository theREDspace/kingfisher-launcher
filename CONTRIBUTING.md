# Contributing

Thanks for your interest in contributing to Kingfisher Launcher!

## How to contribute

1. **Fork the repository** and create a branch from `main`.
2. **Make your changes** following the existing code style and conventions (see [CLAUDE.md](./CLAUDE.md)).
3. **Run the build** to verify everything compiles:
   ```bash
   pnpm build
   ```
4. **Submit a pull request** with a clear description of the change.

## Reporting issues

Open an issue at https://github.com/theREDspace/kingfisher-launcher/issues.

## Code of conduct

Please note that this project follows a [Code of Conduct](./CODE_OF_CONDUCT.md).
By participating, you agree to uphold its terms.
